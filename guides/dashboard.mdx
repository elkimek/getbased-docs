The dashboard is the overview for the current profile. It summarizes what matters now, while the dedicated lenses give you deeper workspaces for labs, genetics, body signals, light, and AI synthesis.

This is the current dashboard model, not a temporary redesign note. Use this page to understand how the app is organized today.

## First visit

Fresh profiles start with an empty dashboard and a five-step guided tour. The tour explains the welcome screen, guided chat, demo profiles, profile switching, and Settings. When the tour finishes, guided chat opens so you can decide what to add first.

The empty state is intentionally chat-first:

- **Start guided chat** is the primary path for new users.
- **Import directly** appears when an AI provider is connected and active.
- Demo profiles show the full dashboard without adding your own data.

You can replay the main app tour from **Settings → Display → Guided Tour**.

## App structure

Desktop uses the sidebar. Smaller screens use bottom tabs and the menu.

| Area | What it is for |
|---|---|
| **Dashboard** | Profile overview and customizable widgets |
| **Labs** | Biomarker charts, tables, heatmaps, categories, and marker detail views |
| **Genome** | Raw DNA import, APOE, mtDNA, and SNP interpretation |
| **Body** | Wearables, manual body metrics, sources, supplements, and cycle tracking |
| **Light** | Sun sessions, therapy devices, indoor light, and measurement tools |
| **Insight** | Current Focus, AI insights, Manage → Context, and next-step surfaces |
| **Recommendations** | A global action plan built from all lenses |
| **Compare dates** | Side-by-side lab date comparison |
| **Correlations** | Marker pair scatter plots and presets |
| **Knowledge Base** | Custom document libraries for AI grounding |
| **Custom markers** | Create biomarkers that are not in the built-in schema |

## Default dashboard widgets

After you add data, the dashboard starts with the widgets most users need first:

- **Biological Coherence** — the top-level Biology Scores widget once the context check and enough markers are available
- **Current Focus** — one synthesized read on the latest data
- **Biological Age** — PhenoAge and Bortz signals when the required markers exist
- **Recommended Next Steps** — top data-linked actions across lenses
- **Current Priority** — the marker most worth checking now
- **Quick Markers** — pinned and priority-ranked marker tiles
- **Biometrics Overview** — selected wearable and manual body metrics
- **Light Today** — current light synthesis
- **Key Trends** — auto-selected markers from the active date range

More widgets are available for Genome, alerts, correlations, notes, supplements, cycle tracking, Manage → Context, light conditions, light sessions, individual markers, and individual Biology Score domains.

See [Biology Scores and Biological Coherence](/guides/biology-scores) for the full scoring and coverage-planning guide.

## Customize the dashboard

Use the dashboard action bar to make the page fit each profile.

- **Customize** turns on layout controls for moving and hiding widgets.
- **Add widget** lets you restore hidden widgets, add marker widgets, or add biometric metrics.
- **Reset layout** restores the default widget set and order.

Dashboard layout is saved per profile, so a demo profile, your own profile, and a client profile can each have different widgets.

## Import from the dashboard

Use the header import button or drop files onto the page. getbased recognizes lab PDFs, lab images, raw DNA files, getbased JSON exports, and database bundles.

On a fresh profile, guided chat is the recommended first path. Direct import remains available when the app has enough AI setup to process files.

## Mobile behavior

On mobile, the dashboard compresses into lens-first navigation. Use the bottom tabs for the main lenses, the menu for additional routes, and the chat button for guided setup or follow-up questions. Widgets and modals are the same data surfaces as desktop, adapted for smaller screens.
